# EcoPower 环保能源

EcoPower 是一个 Slimefun4 附属插件。
你需要安装 Slimefun4 才能让此插件运行。

EcoPower 注重环境友好型的可再生能源。
你可以在下方内容列表中查看所有新增的发电机

## 内容
* 高能太阳能发电机 (全天运作)
* 月光发电机 (仅在夜间工作)
* 高能太阳能头盔
* 蒸汽涡轮发电机 (通过岩浆块产生的水蒸气发电)
* 风力发电机
* 闪电接收器 (可收集闪电的能量并转为电力 -> [Youtube视频](https://youtu.be/mAntFr0c1gg))
